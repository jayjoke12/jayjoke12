# Jay

I built Minecraft systems for Bedrock, Java, Spigot and the occasional interactive web project when I want to work outside the game.

Most of what I make is about changing how players interact with a world, not adding content to it. There’s a difference.

## Right Now
I'm using computer vision to move a Minecraft player's head just like moving a joystick on a controller. 

## Impact that I want to make! (My Statement!)

I want to be able to make Minecraft as accessible to people as possible even to those who struggle with different disabilities.

But it doesn’t have to stop at Minecraft. I want what I build to help people interact with the world more easily, especially those who face barriers most of us don’t think about.
